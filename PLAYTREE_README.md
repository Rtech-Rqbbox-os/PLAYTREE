# PLAYTREE — Awaken. Rebuild. Restore Balance.

A free-to-play multiplayer fantasy adventure RPG built with Python/pygame-ce.

## Features
- 6 Unique Classes: Guardian, Ranger, Mage, Mechanic, Beast Tamer, Shadow Assassin
- 4 Campaign Rounds with unique bosses
- 8+ Epic Boss Fights
- Base Building & Crafting
- Creature Taming & Pet Evolution
- Multiplayer with real-time chat
- Character Customization (hats, auras, trails, titles)
- Battle Pass System
- Full Xbox Controller Support
- Skill Trees & Enchanting
- Weather System (rain, snow, fog)
- Day/Night Cycle
- Fishing Minigame
- RHYSTECH Account System
- Free to Play

## Platforms
- Windows 10/11 (PlayTree.exe)
- Xbox One (coming soon)
- Xbox Series X|S (coming soon)
- Xbox Cloud Gaming (coming soon)

## Download
Download the latest release from [playtree.game](https://playtree.game)

## How to Play
1. Download PLAYTREE.exe
2. Create your RHYSTECH account
3. Choose your class
4. Fight through 4 rounds of enemies and bosses
5. Restore balance to the world

## Tech Stack
- Python 3.14
- pygame-ce 2.5.7
- PyInstaller (packaging)
- Procedural audio (no assets needed)
